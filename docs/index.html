GitHub Pages Setup Guide

Your site is currently being prepared for deployment. Follow these steps to complete the setup:

  1. Build your React app locally: npm install && npm run build
  2. Copy build files to /docs folder
  3. Push to GitHub: git add . && git commit -m "Deploy to GitHub Pages" && git push
  4. Go to Settings → Pages and select /docs as the source
  5. Your site will be live at: https://osaroekaa.github.io/E-commerce-website
